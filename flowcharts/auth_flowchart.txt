LEGEND:
  ( TERMINAL )      = Start / End of a process
  [ PROCESS ]       = An internal action, calculation, or database operation
  / INPUT/OUTPUT /  = Data entry by user (Input) or displaying info/screens (Output)
  < DECISION >      = A conditional check (Diamond)
  ----              = Normal flow arrow
  - - -             = Alternate path / Error loop
